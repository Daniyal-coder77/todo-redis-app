# Todo App with Redis

## Features
- Add, delete, view todos
- Data stored in Redis (Upstash)

## Tech Stack
- Node.js, Express, Redis, HTML/CSS/JS

## Setup locally
1. Clone repo
2. `npm install`
3. Create `.env` file with Upstash credentials
4. `node server.js`
